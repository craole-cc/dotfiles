## Wi-Fi

```sh
passwd
systemctl start wpa_supplicant.service
INTERFACE="wlp4s0"
SSID="ARRIS-7D7B-5G"
SSID_KEY="50A5DC337D7B"
wpa_supplicant -B -i "$INTERFACE" -c <(wpa_passphrase "$SSID" "$SSID_KEY")
ip -a
```
